HỆ THỐNG CDSS BHYT — MÔ TẢ CHỨC NĂNG & CODE HIỆN TẠI
================================================================================

CDSS Bảo hiểm y tế (ung_dung_cdss_bhyt v1.2.1) là hệ thống hỗ trợ ra quyết định
lâm sàng và giám định hồ sơ BHYT theo QĐ 130, với hơn 130 quy tắc kiểm tra trên
6 phân hệ XML. Ứng dụng chạy trên web, mobile (Expo) và desktop Windows
(Electron).

Bản triển khai web dev: https://checkbhytdev.vercel.app
Repo GitHub: https://github.com/htthinh28/check_bhyt
Ngày ghi nhận: 2026-06-29

4. CÁC PHÂN HỆ CHỨC NĂNG
================================================================================

4.1. Xác thực & phân quyền
---------------------------
- Đăng nhập (/login): tài khoản lưu local + Firebase Auth cho cloud
- RBAC (rbac_engine.jsx): vai trò Admin, Doctor, Nurse, Accountant,
  Quality Manager, Head of Dept
- Phân quyền màn hình (/permissions): gating theo coQuyenManHinh()
- Đổi mật khẩu bắt buộc khi tài khoản yêu cầu
- File: ma_nguon/man_hinh/dang_nhap.jsx, phan_quyen_truy_cap.jsx

4.2. Giám định hồ sơ BHYT (lõi hệ thống)
-----------------------------------------
Luồng chính:
  1. Đọc XML (/auditing) — import XML130/4210 từ file
  2. Kiểm tra cấu trúc — schema QĐ 130 (XML1–6)
  3. Engine giám định — dong_co_giam_dinh.jsx chạy 5 lớp quy tắc
  4. Chi tiết ca (/case-detail/:maLK) — xem cảnh báo, lỗi theo từng XML
  5. Sửa XML (/auditing/edit/:maLK) — chỉnh và kiểm tra lại
  6. Kho lưu trữ (/archive) — lưu/truy xuất hồ sơ đã import

5 lớp engine giám định:
  Lớp 0 : Lọc đối tượng BHYT
  Lớp 1 : Hành chính XML1 (HC-01…HC-10, quyền lợi, CV302…)
  Lớp 2 : Danh mục nội bộ ↔ danh mục BYT
  Lớp 3 : Giá thầu, số lượng, chất lượng DM BV
  Lớp 4 : Lâm sàng: thuốc, CDHA, giường, PTTT, chuyển tuyến, ICD-10 TT06…
  Lớp 5 : Quy tắc động (tab), DVKT-OP (VBHN 17), chuyên đề CHUYEN_DE

Validator chuyên biệt: CV302, CV4262, CV3231, CCHN công khám, tương tác thuốc,
phác đồ CDSS, ICD-10 mã kép.

File lõi:
  - ma_nguon/tien_ich/dong_co_giam_dinh.jsx
  - ma_nguon/tien_ich/nhap_file_xml.jsx
  - ma_nguon/tien_ich/kiem_tra_xml.jsx
  - ma_nguon/tien_ich/kho_du_lieu.jsx
  - ma_nguon/tien_ich/dvkt_op_giam_dinh.jsx
  - ma_nguon/tien_ich/quy_tac_on_off_noi_bo.jsx

4.3. Quản lý quy tắc (ON/OFF)
------------------------------
- Quản lý luật (/rules): xem/sửa bộ quy tắc theo mục
- Quy tắc ON/OFF (/rules/on-off): bật/tắt từng MA_LUAT, ghi đè nội dung
- Nguồn quy tắc (ưu tiên): seed bundle → local storage → hardcoded fallback
  → trạng thái ON/OFF người dùng

Seed bundles:
  - du_lieu_luat_du_lieu_muc1.jsx   + seed_luat_du_lieu_muc1.jsx
  - du_lieu_luat_hanh_chinh_muc2.jsx + seed_luat_hanh_chinh_muc2.jsx
  - du_lieu_luat_thuoc_muc8.jsx      + seed_luat_thuoc_muc8.jsx
  - du_lieu_luat_pttt_muc11.jsx      + seed_luat_pttt_muc11.jsx

Hardcoded fallbacks: luat_*_hardcoded.jsx (9 chủ đề)

4.4. Danh mục
--------------
- DM nội bộ (/master-data): nhân sự, thuốc, DVKT, giường, thiết bị (M01–M06)
- DM Bộ Y tế (/danh-muc-byt): 13 phụ lục QĐ 7603 / QĐ 3276
- Mapping nghiệp vụ (/mapping-nghiep-vu): khớp mã BV ↔ BYT, import Excel
- XML chi tiết (/xml/xml1 … /xml/xml6): quản lý từng phân hệ QĐ 130

4.5. Chuyên môn lâm sàng
-------------------------
- Quản lý chuyên môn (/clinical-guidelines): phác đồ BV, quy trình BYT
- Phác đồ CDSS: in ấn, infographic (ma_nguon/chuyen_mon/phac_do_benh_vien/)
- Tương tác thuốc (ma_nguon/chuyen_mon/tuong_tac_thuoc/)
- Phạm vi hành nghề: CCHN, chuyên khoa

4.6. Thư viện & tri thức
-------------------------

Thư viện DVKT (/thuvien/):

  - 3160 quy trình kỹ thuật BYT (QTKT)
  - Dược thư BHYT

4.7. Tích hợp & vận hành
-------------------------
- Cổng HIS (/his-gateway): REST/FHIR/GraphQL/SOAP/WebSocket
